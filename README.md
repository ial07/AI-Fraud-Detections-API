### Quick Integration

```php

$assessment = $api->analyzeTransaction([
    'amount' => 999.99,
    'merchant_id' => 'MERCHANT_123',
    'user_id' => 'USER_456',
    'card_token' => 'token_xxx',
    'ip_address' => '192.168.1.1',
    'user_agent' => 'Mozilla/5.0...'
]);

if ($assessment->risk_score > 0.75) {
    // Block or challenge transaction

    // Approve transaction
}
```
